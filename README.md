# FINANCIAL-RISK-MANAGMENT-PROJECT-
Developed an end-to-end Power BI solution to analyze financial health, track key profitability metrics,
and identify cost-saving opportunities from raw financial datasets
# TECHNICAL WORKFLOW:
# Data Cleaning (Power Query):
Standardized currency formats, handled missing values, and performed data profiling to ensure 100% data integrity.
# Data Modeling: 
Established a Star Schema with Fact and Dimension tables to optimize report performance.
# Python_Statistical_Analysis 
predictive analisis by python libraries.
# Analysis (DAX): 
Created advanced measures for YoY Growth, Profit Margins, and Budget vs. Actual Variance using Time Intelligence functions.
# Visualization: 
Designed interactive dashboards featuring KPI Cards, Trend Lines, and Decomposition Trees for deep-dive analysis.
# Key Impact:
Automated manual financial reporting, reducing processing time by 40%.Provided stakeholders with a "Single Source of Truth" for faster, 
data-driven decision-making.Tech Stack: Power BI (DAX, Power Query), Excel/SQL, Data Modeling.
